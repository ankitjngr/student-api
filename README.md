# 🎓 Student Management API (Django + JWT)

## 📌 Overview
This project is a backend REST API for Student Management system built using Django REST Framework. It includes authentication using JWT and basic CRUD operations for students.

---

## 🚀 Features
- User Signup API
- User Login API (JWT Authentication)
- Secure Token-based Authentication
- Student CRUD Operations (Create, Read, Update, Delete)
- RESTful API structure

---

## 🛠 Tech Stack
- Python
- Django
- Django REST Framework (DRF)
- Simple JWT (Authentication)
